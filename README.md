# EmployeePayRoll
# 💼 Employee Payroll System (Java + JDBC + MySQL)

## 📌 Project Overview
The **Employee Payroll System** is a simple console-based Java application that demonstrates how to connect Java with a MySQL database using JDBC.  
It allows users to manage employee salary records and understand basic database operations in a real-world scenario.

---

## 🚀 Features
- ➕ Add employee details (ID, Name, Basic Salary, Bonus)
- 🧮 Calculate total salary (Basic + Bonus)
- 💾 Store employee records in MySQL database
- 📊 Display payroll report

---

## 🛠️ Tech Stack
- **Java (Core Java)**
- **JDBC (Java Database Connectivity)**
- **MySQL**
